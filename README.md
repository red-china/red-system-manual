### 翻译流程
- 克隆仓库到本地
- 编辑README.md，在要翻译的章节前打勾，以便让他人知道此章已有人在翻译
- 提交README.md至仓库，让他人看到最新的翻译进度
- 在本地进行翻译，翻译好后提交PR即可
